Linux 常用命令总结

1.定期执行脚本

1）crom 时间表：min hour dayofmonth month dayofweek command

每天上午10:15：15 10 command
每周一上午10:15； 15 10 1 command
每月最后一天：00 12 if [ ‘date + %d -d tomorrow’ = 01] ; then ; command

2) crontab -l 列出已有的时间表

3）crontab -e 添加相应任务

4）开启crontab服务：service crond start

5）关闭服务：service crond stop

6）重启服务：service crond restart

7）加入开机自启动：chkconfig -level 35 crond on

2. nohup 不挂断，退出账户时进程还不会结束

3. 处理信号

1 SIGHUP 挂起进程
 2 SIGINT 终止进程
 3 SIGQUIT 停止进程
9 SIGKILL 无条件终止进程
    kill -9 进程号

4.重定向

file1 > file2 file1内容输入到file2

file1 < file2 file2内容输入到file1
